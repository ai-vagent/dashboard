1

대시보드 한눈에 보기

화면은 크게 세 영역으로 구성됩니다.

① 사이드바 (왼쪽)

전체 보기 / 작업자별 필터

  • 본인 이름 클릭 → 본인 알림 페이지
  • 다른 작업자 클릭 → 그 사람의 작품만 보기

② 상단 탭

  • 작품별 현황 — 작품 카드 + 회차 표
  • 주간 타임라인 — 캘린더 위 간트 차트

③ 우측 상단

  • 관리자용 보드 — 전체 통계 페이지
  • 작업자 추가 / 작품 추가 버튼
💡 Tip: 화면 변경사항은 모두 자동 저장되며, 다른 사람도 실시간으로 같은 화면을 봅니다.
2

자기 페이지에서 받는 알림

사이드바에서 본인 이름을 클릭하면 해야 할 일 두 가지가 카드로 나타납니다.

🔔 컨펌 필요

작업자가 회차 단계를 영상초안작업 · 피드백 수정중 · 마무리 작업중 중 하나에서 진행률 100%로 만들면 자동 발생.

→ 이채은, 배종일 모두에게 표시

✏️ 각색 필요 이채은 전용

어떤 회차가 미시작인데 직전 회차가 피드백 수정중 이상이면 자동 발생.

예: 살아남은 로맨스 1화가 피드백 수정중 → 2화 각색 필요 알림

💡 사이드바 본인 이름 옆 🔔 숫자가 처리해야 할 알림 총 개수입니다.
3

컨펌 카드 — '완료' 버튼으로 다음 단계 진행

살아남은 로맨스 3화
김여은
5/18~5/22
영상초안작업
완료

완료 버튼을 누르면…

영상초안 100%
→
완료 클릭
→
피드백 수정중 0%로 진행
→

⚠️ 주의: '완료'를 누르면 단계가 자동으로 진행됩니다. 실수로 누른 경우 작업자가 회차 모달에서 단계를 되돌릴 수 있습니다.
4

각색 카드 — 이채은 전용

다음 회차 각색을 시작해야 한다는 알림. '완료'를 누르면 회차가 미시작 → 각색 완료 단계로 진행됩니다.

김여은

2화 미시작
→
이채은이 완료 클릭
→
2화 각색 완료 단계로 진행
→
김여은 화면에 ✓ 각색 완료
5

작품별 현황 탭

상단의 알림 카드 아래 자기 담당 작품이 카드로 표시됩니다.

전처리 단계 작품

작품 카드에 전처리 진행 단계 (작품 선정 → 아트스타일 → 주요 인물)가 표시됩니다.

연재 진행 중 작품

전처리가 끝난 작품은 회차 표가 표시되어 각 회차의 진행 상황을 확인 / 수정할 수 있습니다.

회차 표에서 가능한 작업

  • 회차 행 클릭 → 단계 / 진행률 / 링크 수정 모달
  • 일정 셀 클릭 → 캘린더 팝업으로 시작일 / 종료일 변경
  • 완료 및 업로드 회차는 '링크' 칸에 결과물 링크 입력 → 표에는 🔗 아이콘으로 표시(클릭 시 새 탭)
  • 회차가 많으면 10화 단위 페이지네이션 (표 하단 ‹ 이전 / 다음 ›)
  • 이번 주 회차는 보라색 강조 + "이번 주" 라벨
6

주간 타임라인 탭

캘린더 위에 회차 작업 기간이 막대로 표시되는 간트 차트 형식

본인 행 (상단)

  • 컨펌 필요 막대 — 모든 컨펌 항목
  • 각색 필요 막대 — 이채은 전용

작업 현황 (작품별 행)

  • 각 회차의 시작일~종료일 막대
  • 단계별 색상 / 진행률 게이지
  • 오늘 위치는 보라색 세로선으로 표시
💡 막대 클릭으로 회차 단계 수정 모달이 열립니다.
7

관리자용 보드 — 작업자(제작일정) / 작품 탭

'관리자용 보드'가 두 개의 작은 서브탭으로 나뉩니다. 상단의 작은 탭으로 전환하세요.

🗓️ 작업자 (제작일정) 기본 탭

  • 📊 제작 일정 인사이트 — 요약 스탯(평균 제작일수·총 지연 영업일·지연 최다 단계), 프로세스(단계)별 지연, 작품별 평균 제작일수
  • ⚠️ 일정 지연 회차 목록 + 지연 일수
  • 작업자별 완료 회차 작업일 — 작품별 기준일과 비교해 초과 / 단축 표시

📚 작품

  • 전체 작품 현황 · 전체 회차 진행률(% + 진행 바)
  • 📺 연재중 · ⏳ 대기중 · ✅ 완료 · 👤 담당자 미정 표
  • 현황 카드 클릭 → 해당 카테고리 표로 스크롤
⚠️ 제작 일정 인사이트 안내: 회차는 단계 전환 시각을 따로 기록하지 않아 '단계별 지연'은 각 회차의 현재 단계 기준으로 집계한 근사치입니다. 완료 · 지연 회차가 쌓일수록 정확해집니다.
8

작품 / 작업자 추가

+ 작품 추가

  • 작품명, 메인 작업자(미정 가능), 시작월, 회차수, 단계 입력
  • 저장 시 회차가 월~금 5일 일정으로 자동 생성
  • 담당자가 정해지지 않은 작품은 전체 보기 하단에 별도 표시

+ 작업자 추가

  • 이름, 역할(감독/일반), 부역할, 색상
  • 감독은 사이드바 상단, 일반 작업자는 하단에 추가됨
💡 작품 수정: 작품 카드 우측의 ✎ 아이콘을 클릭. 회차 수나 시작월을 바꾸면 회차가 다시 생성되어 진행 상태가 초기화됩니다 — 주의.
9

일정 조정 — 자동 캐스케이드

예시: 살아남은 로맨스 1화 일정을 5/4~5/8 → 5/4~5/11로 변경 (1 영업일 추가)
