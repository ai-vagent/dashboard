1

첫 화면 — 본인 이름 클릭

사이드바에서 자기 이름을 클릭하면 본인 담당 작품만 보입니다.

① 작품별 현황

  • 각 작품별 카드 + 회차 표
  • 회차별 단계 / 진행률 / 링크 확인 / 수정
  • 회차가 많으면 10화씩 페이지로 나뉨
  • 이번 주 회차는 보라색 강조

② 주간 타임라인

  • 캘린더 위 회차 막대 (간트 차트)
  • 막대 색 = 단계, 게이지 = 진행률
  • 오늘은 보라색 세로선으로 표시
💡 화면 변경은 자동 저장 — 별도 저장 버튼 없음. 다른 사람도 실시간으로 같은 화면을 봅니다.
2

회차 작업 7단계

단계는 순서대로 진행합니다. 진행률(%)은 일부 단계에서만 사용됩니다.

0미시작
1각색 완료
2영상 추출 중
3영상초안작업
4피드백 수정중
5마무리 작업중
6완료 및 업로드

진행률(%) 사용 단계

영상초안작업 · 피드백 수정중 · 마무리 작업중

이 단계에서 100%가 되면 자동으로 '🔔 컨펌 필요' 알림이 감독에게 전송됩니다.

각색 / 영상 추출 단계

각색 완료는 이채은이 '완료' 처리하면 자동 진입.

영상 추출 중부터는 작업자가 직접 단계를 진행시킵니다.

6

일정 변경 — 캘린더에서 직접

회차 표의 '일정' 셀을 클릭하면 캘린더 팝업이 열립니다.

선택 방법

  • 첫 번째 클릭 → 시작일
  • 두 번째 클릭 → 종료일
  • 역순 클릭 시 자동 스왑
  • 토·일은 선택 불가 (휴무)

⚠️ 자동 시프트

앞 회차 일정을 늘리면 뒤 회차들의 일정이 영업일 차이만큼 자동으로 밀립니다.

예: 1화를 +1 영업일 늘리면, 2화 / 3화… 모두 +1 영업일씩 시프트

⚠️ 일정 지연 자동 추적: 원본 일정은 변하지 않고 별도 저장됩니다. 일정을 늘릴 때마다 관리자용 보드의 '일정 지연' 섹션에 자동으로 누적됩니다.
7

'작품별 현황' vs '주간 타임라인'

상황에 따라 적절한 뷰를 골라 사용하세요.

📋 작품별 현황

이럴 때 사용:

  • 회차 단계나 진행률을 자주 수정할 때
  • 링크(결과물) / 메모를 입력·확인할 때
  • 특정 작품의 전체 회차를 한눈에 볼 때

📊 주간 타임라인

이럴 때 사용:

  • 전체 작품의 일정 흐름을 시각적으로 볼 때
  • 이번 주 / 다음 주 작업 범위를 확인할 때
  • 일정 지연 / 겹침을 점검할 때
💡 막대 / 행 클릭으로 회차 수정 모달이 바로 열립니다 — 두 뷰 모두에서 가능.
8

실시간 협업 / 모바일 지원

자동 저장

변경 즉시 클라우드(DB)에 저장. 별도 저장 버튼 / 새로고침 불필요.

실시간 동기화

감독 / 다른 작업자의 변경이 1초 내에 내 화면에도 반영.

모바일 / 태블릿

같은 URL을 휴대폰 브라우저로 열면 자동으로 모바일 레이아웃. 모든 기능 동일하게 동작.

💡 모바일에서는 좌측 상단 ☰ 버튼으로 사이드바를 열어 작업자 선택.
